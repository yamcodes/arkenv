## Why use the CLI?

While you can always set up ArkEnv manually, the CLI ensures that you:

- Use the correct import paths for your chosen validation library.
- Follow best practices for project structure.
- Have a working, typesafe environment configuration in seconds.

Next, you might want to learn about [how to use ArkEnv](/docs/arkenv) or check out our [Vite Plugin](/docs/vite-plugin) and [Bun Plugin](/docs/bun-plugin) for build-time validation.
