ArkEnv for Nuxt supports any validator that implements the [Standard Schema](https://standardschema.dev/) specification — including Zod, Valibot, Typia, and others listed here:

<Cards>
  <Card title="Which schema libraries implement the spec?" href="https://standardschema.dev/schema#what-schema-libraries-implement-the-spec" description="Standard Schema" />
</Cards>

## Standard Mode (no ArkType)

Use `@arkenv/standard` and the `@arkenv/nuxt/standard` subpaths when you want Zod, Valibot, or another Standard Schema validator **without** installing `arktype`.

### Install

```package-install
@arkenv/standard @arkenv/nuxt zod
```

### Register the Standard Mode module

Use `@arkenv/nuxt/standard/module` instead of `@arkenv/nuxt/module`:

```ts title="nuxt.config.ts"
export default defineNuxtConfig({
	modules: ["@arkenv/nuxt/standard/module"],
});
```

### Define a flat schema

Import from `@arkenv/nuxt/standard` and use a flat object (same shape as the [flat layout](/docs/nuxt/layouts/flat)):

```ts title="env.ts" twoslash
import arkenv from "@arkenv/nuxt/standard";
import { z } from "zod";

export const env = arkenv({
	DATABASE_URL: z.string().url(),
	NUXT_PUBLIC_API_URL: z.string().url(),
	NODE_ENV: z.enum(["development", "production", "test"]).default("development"),
});
```

:::tip
Prefer the [ArkEnv CLI](/docs/cli) (`npx arkenv@latest init`) and pick Zod or Valibot — it scaffolds Standard Mode for you.
:::

:::note
Legacy nested layout `arkenv({ server, client, shared })` is still supported but deprecated. Prefer the [flat layout](/docs/nuxt/layouts/flat).
:::

## Mixing with ArkType

Already on `@arkenv/core`? You can mix ArkType DSL strings with Zod (or any Standard Schema validator) in the same schema — ArkType is still required as a peer dependency:

```ts title="env.ts" twoslash
import arkenv from "@arkenv/nuxt";
import { z } from "zod";

export const env = arkenv({
	HOST: "string.host",
	PORT: "number.port",
	DATABASE_URL: z.string().url(),
	NUXT_PUBLIC_API_URL: z.string().url(),
});
```

For the ArkType-free path, stay on [Standard Mode](#standard-mode-no-arktype) above. For core (non-Nuxt) mixing examples, see [Standard Schema validators](/docs/arkenv/integrations/standard-schema).
