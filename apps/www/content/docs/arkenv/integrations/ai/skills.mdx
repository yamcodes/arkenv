ArkEnv provides official **Agent Skills** to help AI coding agents (like Cursor, Claude Code, Windsurf, and Gemini) understand and use the library more effectively.

Skills provide agents with on-demand procedural knowledge, allowing them to write schemas, handle complex types, and configure integrations with high precision.

## Installation

You can add ArkEnv skills to your project using the [Agent Skills CLI](https://skills.sh):

```bash
npx skills add yamcodes/arkenv --all
```

## Available Skills

### `arkenv`

The core skill for library usage. It helps agents:

- Define typesafe schemas using ArkType notation.
- Implement complex types and arrays.
- Configure automatic coercion and default values.
- Follow best practices for schema organization.

### `arkenv-cli`

The setup and DevOps skill. It helps agents:

- Initialize ArkEnv in new or existing projects.
- Scaffold schema files.
- Automatically configure `tsconfig.json` for optimal typesafety.

## When to use

These skills are particularly useful when:

- You are using **Cursor** or **Windsurf** as your primary IDE.
- You want **Claude Code** to help you refactor or set up your environment variables.
- You want to ensure AI-generated schemas strictly follow ArkEnv's recommended patterns.

For more information about the skills ecosystem, visit [skills.sh](https://skills.sh).
