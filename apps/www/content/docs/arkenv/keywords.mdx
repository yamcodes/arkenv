---
title: Keywords
description: ArkEnv keywords that extend ArkType for common environment-variable shapes.
---

ArkEnv adds a small set of env-oriented keywords on top of ArkType. You can reference them in any ArkEnv schema or `type` definition:

:::note
**Only works with ArkType.** Keywords are not available from [`arkenv/standard`](/docs/arkenv/standard) - import from `arkenv` when you need `string.host` or `number.port`.
:::

```ts twoslash
import arkenv, { type } from "arkenv";

const env = arkenv({
  HOST: "string.host",
  PORT: "number.port = 3000",
});

// Hover Host / Port to see the inferred types
const Host = type("string.host");
type Host = typeof Host.infer;
const Port = type("number.port");
type Port = typeof Port.infer;
```

:::note
The full [ArkType primitives](https://arktype.io/docs/primitives) keyword set still works in ArkEnv schemas. This page documents only the keywords ArkEnv adds.
:::
