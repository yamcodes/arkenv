## AI Skills

ArkEnv provides official [Agent Skills](https://skills.sh) to help AI coding agents (like Cursor, Claude Code, Windsurf, and Gemini) understand and use ArkEnv more effectively.

```bash
npx skills add yamcodes/arkenv --all
```
